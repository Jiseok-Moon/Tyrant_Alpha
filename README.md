사용 엔진 및 기술 스택
Engine : Unity 2022.3
Language : C#
Key Systems : Ui시스템(아이템, 일시정지, 게임오버 패널) 등

핵심 구현 정보
Project Tyrant 기획서(Google 드라이브 내 존재)의 플레이어블 캐릭터 '혈마법사' 구현
캐릭터 컨셉에 맞춘 자원 소모(HP) 및 유기적 회복 메커니즘 설계 및 구현
개성을 살린 기본 스킬 5종의 로직 개발

전투 및 상호작용
적 유닛과의 피격/타격 판정 등 상호작용 시스템 구현
게임의 톤앤매너를 반영한 스킬 이펙트 및 아이콘 리소스 적용

게임 루프 및 UI
인벤토리(아이템), 게임 흐름 제어(일시정지), 종료 창(게임오버) 등 필수 게임 내비게이션 Ui 연동
